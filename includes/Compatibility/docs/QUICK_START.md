## How It Works

### For Payment Gateways

Payment gateways that support WooCommerce Subscriptions will now work with WPSubscription:

1. **Gateway calls WCS functions** â†’ Compatibility layer intercepts
2. **Compatibility translates to WPS** â†’ Uses WPSubscription data
3. **Returns WCS-compatible format** â†’ Gateway works seamlessly

### Available Functions

All standard WooCommerce Subscriptions functions are available:

```php

// Get subscription by ID

// Check if product is subscription
```
